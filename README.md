# ✅ Taskiva - Todo App

Taskiva is a simple and user-friendly Todo Application that helps users organize daily tasks, improve productivity, and manage routines efficiently.

## 🚀 Live Demo

🔗 https://taskiva-todo-app.vercel.app

## 📌 Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clean and responsive user interface
- Easy task management
- Works directly in the browser

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Vercel (Deployment)

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/NarayanSharma123/Todo-App.git
```

### 2. Navigate to the Project Folder

```bash
cd Todo-App
```

### 3. Run the Application

Simply open the `index.html` file in your browser.
